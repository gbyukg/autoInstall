使用说明

./install.sh argv1 argv2 argv3 argv4 argv5 argv6 argv7 argv8

参数说明:

  • argv1:选择安装sugar方式，接受参数：git(git下载安装方式)/url(通过url下载指定版本方式)
    • git:选择要进行安装的主分支名
    • url:要下载的文件的url后面部分信息
    • git:要进行合并的分支名，默认为0，表示不合并分支
    • url:要下载的文件名，不包含文件的扩展名(.zip)
  • argv4:数据库名,默认sugar
  • argv5:数据库主机名，默认localhost
  • argv6:数据库端口，默认50000
  • argv7:数据库用户名，默认db2inst1
  • argv8:数据库密码，默认admin

注意事项

只能在当前目录下使用./install.sh或使用绝对路径/aaa/bbb/ccc/install.sh方式执行此脚本，不能使用bash、sh、source等方式执行，否则路径会有错误。

需要安装expect来支持切换数据库实例帐号后的数据库初始化工作，ubuntu安装方式：

$
sudo apt-get install expect

sugar安装过程采用PHP方式，需要PHP支持curl扩展功能

补充

由于脚本中调用了expect脚本，在页面输出终端内容会有问题，一直无法解决。因此暂时无法集成到jenkins或janky中，当前所使用到的文件只有install.sh、install.php和initdb.exp三个文件。
